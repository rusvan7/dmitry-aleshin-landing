# Dmitry Aleshin Landing

Static landing prototype for GitHub Pages.

## Publish to GitHub Pages

1. Create an empty GitHub repository, for example `dmitry-aleshin-landing`.
2. Push this folder to the repository's `main` branch.
3. In GitHub, open `Settings -> Pages`.
4. Set source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.

The site URL will look like:

```text
https://USERNAME.github.io/dmitry-aleshin-landing/
```
